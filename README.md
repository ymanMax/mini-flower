# mini-flower
专注于鲜花销售的微信小程序，提供各类鲜花展示、在线购买、订单管理等功能，让用户轻松选购心仪鲜花。

## 功能特性

### 鲜花展示
- **首页轮播图**：展示热门鲜花和促销活动
- **鲜花推荐**：根据用户喜好推荐相关鲜花
- **分类浏览**：按鲜花分类进行浏览，支持多级分类
- **鲜花搜索**：支持根据关键词搜索鲜花
- **鲜花详情**：展示鲜花的详细信息，包括价格、规格、配送信息等

### 购物车功能
- **添加商品**：支持将鲜花添加到购物车
- **购物车管理**：支持修改商品数量、删除商品
- **价格计算**：自动计算购物车中商品的总价
- **本地缓存**：购物车数据保存在本地缓存中，刷新页面不会丢失

### 订单管理
- **创建订单**：支持将购物车中的商品生成订单
- **订单支付**：支持在线支付订单（集成微信支付）
- **订单列表**：支持查看所有订单，包括待付款、待发货、待收货、已完成等状态
- **订单详情**：支持查看订单的详细信息，包括商品列表、价格、地址等

### 个人中心
- **用户信息**：支持查看和修改用户信息
- **订单管理**：支持查看和管理用户的所有订单
- **地址管理**：支持查看和管理用户的收货地址
- **收藏夹**：支持查看和管理用户收藏的鲜花
- **浏览历史**：支持查看用户的浏览历史

## 技术栈

- **开发框架**：微信小程序原生框架
- **数据请求**：使用wx.request进行API请求
- **状态管理**：使用微信小程序的globalData进行全局状态管理
- **本地存储**：使用wx.setStorage和wx.getStorage进行本地数据存储
- **UI设计**：使用微信小程序原生组件进行UI设计，风格简洁美观
- **支付功能**：集成微信支付SDK，支持在线支付

## 使用方法

### 1. 环境准备

- 下载并安装微信开发者工具
- 注册微信小程序账号（用于发布和支付功能）
- 配置微信支付功能（需要企业资质）

### 2. 项目运行

- 克隆或下载本项目代码
- 打开微信开发者工具，导入本项目代码
- 配置小程序AppID和AppSecret
- 点击“编译”按钮，即可运行本项目

### 3. 项目配置

- **API接口配置**：在各页面的JS文件中配置API请求路径
- **全局样式配置**：在`/app.wxss`文件中配置全局样式
- **页面配置**：在各页面的`json`文件中配置页面样式和功能
- **支付配置**：在`/app.js`文件中配置微信支付参数

## 注意事项

- 本项目使用真实的API接口，需要确保网络连接正常
- 支付功能需要企业资质和微信支付配置，个人开发者无法使用
- 项目中使用了本地缓存，需要注意数据的一致性
- 项目中包含多个页面和组件，需要根据实际需求进行使用和配置
